# Client demonstration script

1. Open Command Center; expand and pin a notice and filter Knowledge Center.
2. Open Crop Protection Orders; search, switch tabs/page size, and export CSV.
3. Create an order and select MFA-ADRIAN; point out populated Sold-To, email, and shipping instructions.
4. Add Products; search Product Name for `sterling blue`, choose an exact SKU, and set quantity.
5. Search Active Ingredient for `glyphosate` and show multiple suppliers.
6. Add products, inspect totals, save the draft, return to Draft Orders, and resume it.
7. Review validation and predicted grouping; correct any blocking issue.
8. Confirm submission, inspect the Web Order Number, and download the PDF.
9. Return to Placed Orders and locate the submission.
10. Open the assistant; ask for Sterling Blue, supply missing account/SKU details, and explain that it uses the same catalog and order service boundary.
11. Set `MockJde__FailSubmissions=true`, restart the API, and show that failure never displays false success and the order remains recoverable.
